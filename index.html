A propos

Qui je suis ?

Bonjour,

Je m'appelle Sami et je suis passionné par l'informatique et la programmation.

Cette passion m'a conduit à me former intensivement. Tout d'abord, j'ai suivi le programme "Développeur Intégrateur Web" de l'IFOCOP, puis la formation "Développeur Concepteur Python".

Grâce à ces formations, j'ai pu développer des applications logicielles et web en utilisant diverses technologies, notamment : Python, Php, Javascript, HTML/CSS, Bootstrap, jQuery et SQL.

Ma spécialisation en programmation Python m'a permis de maîtriser plusieurs packages, tels que : Django, Flask, Beautiful Soup, Pandas, Django rest Framework, Pytest et Flake8.

J'ai également acquis les compétences suivantes :

  • - Versionner mon code avec Git et GitHub
  • - Créer un environnement virtuel et installer les dépendances nécessaires à une application
  • - Garantir la qualité du code en effectuant les tests nécessaires, ainsi que le formatage et le linting
  • - Mettre en place un processus ETL pour scraper les données d'une page web
  • - Mettre en place un pipeline CI/CD avec GitLab
  • - Déployer une application avec Render
  • - Conteneuriser une application avec Docker et l'héberger sur DockerHub

Plus qu'un métier, c'est une véritable passion qui m'anime.

Je suis rigoureux, sérieux, perfectionniste, mais surtout passionné : ce sont ces qualités que je souhaite mettre à contribution pour vous aider à réaliser votre projet
